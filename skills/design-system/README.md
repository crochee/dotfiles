# Design System

Generate or audit design systems, extract design tokens, and detect AI-generated design patterns.

## What This Skill Does

Analyzes codebases for visual consistency, generates design token sets, scores UI across 10 dimensions, and detects generic "AI slop" patterns.

## When It Activates

- Starting a new project needing a design system
- Auditing existing UI consistency
- Reviewing PRs that touch styling
- Detecting generic AI-looking UI patterns

## Key Topics Covered

- Mode 1: Generate design system from existing code
- Mode 2: Visual audit with 10-dimensional scoring
- Mode 3: AI slop detection
- Design token extraction (colors, typography, spacing)
