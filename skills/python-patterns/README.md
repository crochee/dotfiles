# Python Patterns

Pythonic idioms and best practices for building robust applications.

## What This Skill Does

Provides conventions for Python development including type hints, dataclasses, decorators, context managers, async patterns, and PEP 8 compliance.

## When It Activates

- Writing new Python code
- Reviewing or refactoring Python code
- Designing Python packages
- Implementing type hints, decorators, context managers, or async patterns

## Key Topics Covered

- Type hints (function signatures, generics, protocols)
- Dataclasses and validation
- Decorators and context managers
- Async/await patterns
- Package organization (src layout)
- Memory and performance optimization
- Anti-patterns to avoid
