# Python Testing

Python testing strategies using pytest with TDD methodology.

## What This Skill Does

Provides comprehensive testing patterns for Python including pytest fixtures, mocking, parametrization, async testing, coverage, and CI/CD integration.

## When It Activates

- Writing Python tests with pytest
- Setting up test infrastructure
- Creating test fixtures and mocking external dependencies
- Parametrizing tests
- Testing async code

## Key Topics Covered

- TDD workflow for Python
- pytest fixtures and autouse
- Mocking with unittest.mock and pytest-mock
- Parametrization strategies
- Async testing with pytest-asyncio
- Coverage reporting
- CI/CD integration
- Fast test patterns
