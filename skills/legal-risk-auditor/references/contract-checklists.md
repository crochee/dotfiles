# 合同审查清单 - Contract Review Checklists

5类常见合同的详细审查要点，逐项检查以识别潜在风险。

## 一、买卖合同

- [ ] **标的物描述**: 规格、数量、质量标准是否明确（避免"按样品"等模糊表述）
- [ ] **交付条款**: 交付时间、地点、方式是否具体（逾期交付的违约责任是否约定）
- [ ] **付款条款**: 付款条件、期限、方式是否对等（先货后款/先款后货的风险分配）
- [ ] **验收条款**: 验收标准是否可操作（异议期限、不合格品处理方式）
- [ ] **违约责任**: 是否双向约束（仅约束买方或卖方为常见风险）
- [ ] **质量保证**: 质保期、质保范围、质保责任是否明确
- [ ] **风险转移**: 标的物毁损灭失的风险转移时点是否约定
- [ ] **不可抗力**: 不可抗力的范围、通知义务、免责条件是否合理

## 二、租赁合同

- [ ] **租赁物描述**: 名称、位置、面积、用途是否明确
- [ ] **权属确认**: 出租人对租赁物是否享有合法出租权（产权证明/转租权）
- [ ] **租赁期限**: 起止日期是否明确，是否超过法定上限（《民法典》第705条：不得超过20年）
- [ ] **租金条款**: 金额、支付方式、支付时间、调整机制是否约定
- [ ] **押金条款**: 金额、用途、退还条件、扣除条件是否具体
- [ ] **维修责任**: 日常维修与重大维修的责任分配是否合理
- [ ] **转租/转让**: 是否允许转租、转让的条件是否约定
- [ ] **提前解除**: 解除条件、通知期限、违约责任是否对等
- [ ] **续租优先权**: 是否约定承租人的优先承租权

## 三、服务/委托合同

- [ ] **服务范围**: 包含和不包含的内容是否明确（避免"等相关服务"等兜底表述）
- [ ] **服务标准**: 是否可量化/可验证（SLA、KPI、交付物标准）
- [ ] **交付时间**: 里程碑、最终交付日期、延期处理
- [ ] **验收机制**: 验收标准、验收流程、异议期、默认验收条件
- [ ] **付款节点**: 与服务进度/验收结果挂钩的付款安排
- [ ] **知识产权**: 成果的知识产权归属（委托方/受托方/共有）
- [ ] **保密义务**: 是否双向、保密范围、保密期限、违约责任
- [ ] **竞业限制**: 受托方在合作期间/之后的竞业限制
- [ ] **单方解除权**: 双方解除权是否对等（仅甲方有权解除为常见风险）
- [ ] **责任上限**: 受托方的责任上限是否合理（通常为合同金额的特定倍数）

## 四、劳动/雇佣合同

- [ ] **试用期**: 期限是否符合法定上限（《劳动合同法》第19条）
  - 合同期 < 3个月：不得约定试用期
  - 3个月 ≤ 合同期 < 1年：试用期 ≤ 1个月
  - 1年 ≤ 合同期 < 3年：试用期 ≤ 2个月
  - 合同期 ≥ 3年或无固定期限：试用期 ≤ 6个月
- [ ] **薪酬结构**: 基本工资、绩效工资、津贴补贴、发薪日期
- [ ] **工作时间**: 标准工时/综合工时/不定时工时的约定
- [ ] **加班规定**: 加班审批流程、加班费计算标准
- [ ] **社保公积金**: 缴纳基数、比例、开始时间
- [ ] **竞业限制**: 范围、期限（≤2年）、经济补偿标准（离职后按月支付）
- [ ] **保密义务**: 商业秘密的范围、保密期限、违约责任
- [ ] **合同解除**: 双方解除条件、经济补偿金计算
- [ ] **培训服务期**: 专项培训费用对应的服务期约定
- [ ] **违约金**: 仅限服务期违约和竞业限制违约可约定（《劳动合同法》第25条）

## 五、借款/担保合同

- [ ] **借款金额**: 币种、数额（大写+小写）
- [ ] **借款利率**: 年利率是否超过法定上限（不超过合同成立时LPR的4倍）
- [ ] **借款期限**: 起止日期、展期条件
- [ ] **还款方式**: 一次性还本付息/等额本息/等额本金/先息后本
- [ ] **提前还款**: 是否允许、提前还款通知期、是否有违约金
- [ ] **逾期责任**: 逾期利率、罚息计算方式、催收费用承担
- [ ] **担保方式**: 保证（一般/连带）/抵押/质押/留置
- [ ] **担保范围**: 本金、利息、违约金、损害赔偿金、实现债权的费用
- [ ] **担保期间**: 起算日、期限（未约定的为债务履行期届满后6个月）
- [ ] **抵押/质押登记**: 是否要求办理登记、登记期限、未登记的后果
- [ ] **借款人陈述与保证**: 财务状况真实性、无重大诉讼、用途合法性
- [ ] **加速到期条款**: 触发加速到期的条件是否合理
