# Rust Testing

Rust testing patterns following TDD methodology.

## What This Skill Does

Provides comprehensive testing patterns for Rust including unit tests, integration tests, async testing, property-based testing, mocking, and coverage.

## When It Activates

- Writing Rust tests
- Adding test coverage to Rust code
- Creating benchmarks
- Implementing property-based tests with proptest
- Mocking with mockall

## Key Topics Covered

- TDD workflow for Rust
- Unit tests vs integration tests
- Async testing with tokio-test
- Property-based testing with proptest
- Mocking with mockall
- Doctest documentation
- Coverage reporting
- Benchmarking with criterion
