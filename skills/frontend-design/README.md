# Frontend Design

Create distinctive, production-grade frontend interfaces with high design quality.

## What This Skill Does

Guides the creation of visually compelling, intentional frontend designs that avoid generic AI-looking UI.

## When It Activates

- Building landing pages, dashboards, or app surfaces from scratch
- Upgrading bland interfaces into something intentional and memorable
- Translating product concepts into concrete visual directions
- Implementing frontends where typography, composition, and motion matter

## Key Topics Covered

- Visual direction selection (minimal, editorial, industrial, luxury, etc.)
- Building visual systems (typography, color, spacing, motion)
- Intentional composition and layout
- Meaningful animation and motion design
- Anti-patterns to avoid (generic SaaS heroes, card piles, random accents)
