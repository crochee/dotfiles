# Git Workflow

Git workflow patterns for branching strategies, commit conventions, and collaborative development.

## What This Skill Does

Provides best practices for Git version control, branching strategies, commit message conventions, merge vs rebase decisions, and conflict resolution.

## When It Activates

- Setting up Git workflow for a new project
- Choosing branching strategies (GitHub Flow, Trunk-Based, GitFlow)
- Writing commit messages and PR descriptions
- Resolving merge conflicts
- Managing releases and version tags

## Key Topics Covered

- GitHub Flow, Trunk-Based Development, GitFlow
- Conventional Commits format
- Merge vs rebase decision guide
- Pull request templates and code review checklists
- Conflict resolution strategies
- Branch naming conventions and cleanup
- Semantic versioning and release management
- Git configuration and useful aliases
