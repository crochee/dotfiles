# Deployment Patterns

Production deployment workflows and CI/CD best practices for web applications.

## What This Skill Does

Provides deployment strategies, Docker containerization patterns, CI/CD pipeline templates, health check implementations, and production readiness checklists.

## When It Activates

- Setting up CI/CD pipelines
- Dockerizing applications
- Planning deployment strategies (blue-green, canary, rolling)
- Implementing health checks and readiness probes
- Preparing production releases

## Key Topics Covered

- Rolling, blue-green, and canary deployment strategies
- Multi-stage Dockerfiles for Node.js, Go, Python
- GitHub Actions CI/CD pipeline templates
- Health check endpoints and Kubernetes probes
- Environment configuration (Twelve-Factor App)
- Rollback strategies and production readiness checklists
