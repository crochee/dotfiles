# Writing Plans

Break approved designs into granular implementation tasks.

## Purpose

Transforms high-level design into bite-sized, executable tasks that can be implemented and verified independently. Each task is small enough (2-5 minutes) to complete without losing focus.

## Features

- **Bite-Sized Tasks**: 2-5 minutes each
- **Exact File Paths**: No vague references
- **Complete Code**: Full implementations, not pseudocode
- **Clear Verification**: How to know it works
- **Dependency Ordering**: Tasks ordered by dependencies

## When to Use

- After design approval (brainstorming complete)
- Before implementation begins
- Breaking complex work into manageable pieces

## Output

- Plan saved to `.agents/plan.md`
- Task list with file paths, code, and verification steps
- Estimated total time
- Parallelization notes
