# Frontend Patterns

React and Next.js development patterns for building performant user interfaces.

## What This Skill Does

Provides idiomatic patterns for React components, state management, performance optimization, forms, accessibility, and animations.

## When It Activates

- Building React components
- Managing application state (useState, useReducer, Context)
- Implementing data fetching patterns
- Optimizing frontend performance
- Working with forms and validation

## Key Topics Covered

- Component composition and compound components
- Custom hooks patterns
- State management (Context + Reducer)
- Performance optimization (memoization, virtualization, code splitting)
- Form handling with validation
- Error boundaries
- Animation patterns with Framer Motion
- Accessibility patterns (keyboard navigation, focus management)
