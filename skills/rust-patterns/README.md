# Rust Patterns

Idiomatic Rust patterns for building safe, performant applications.

## What This Skill Does

Provides conventions for Rust development including ownership patterns, error handling, traits, concurrency, and performance optimization.

## When It Activates

- Writing new Rust code
- Reviewing or refactoring Rust code
- Designing crate structures
- Implementing ownership/borrowing patterns
- Working with async Rust

## Key Topics Covered

- Ownership and borrowing patterns
- Error handling (Result, thiserror, anyhow)
- Trait design and implementation
- Concurrency patterns (Send, Sync, Arc, channels)
- Async patterns
- Performance optimization (zero-copy, SIMD)
- Anti-patterns to avoid
